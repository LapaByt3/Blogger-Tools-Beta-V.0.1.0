# Athenix Beta-V.0.1.0

AI-powered content creation tool built with Python and DeepSeek API. Helps bloggers and content creators generate high-quality articles faster.

## ✨ Features

- 📄 **Paraphrase Artikel** - Rewrite content with natural language
- ✨ **Auto Judul Menarik** - Generate 35 unique titles per category
- 🚀 **Generate dari Keyword** - Create in-depth articles (1500-2000 words)
- 🎥 **YouTube ke Artikel** - Convert YouTube videos to articles (2000-3000 words)
- 🔍 **Cek Rank Artikel** - SEO analysis with real-time traffic graphs

## 🛠️ Tech Stack

- Python 3.13.12
- CustomTkinter (UI)
- DeepSeek AI API
- BeautifulSoup4 (Web scraping)
- Matplotlib (Data visualization)

## 🚀 Quick Start

1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Get API Key from [DeepSeek Platform](https://platform.deepseek.com)
4. Run: `python blogger.py`
